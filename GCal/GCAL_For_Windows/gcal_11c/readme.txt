
GCal, developed by the GBC Vaisnava Calendar Committee of the 
International Society for Krishna Consciousness (ISKCON), 
is a free Windows standalone desktop program for calculating 
calendars according to the Gaudiya Vaisnava system. This is the 
eleventh publicly available version of GCal, released for download 
on 21st August 2011. For program modifications since the earlier 
versions, see the GCal Release History file.


==============================================
    GCAL  Quick Installation
==============================================

1. unzip gcal.zip file
2. copy files gcal.exe and gcal.chm to your local drive
3. gcal.exe and gcal.chm has to be in the same directory
4. run gcal
